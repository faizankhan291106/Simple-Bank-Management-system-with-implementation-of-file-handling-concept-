# 🏦 Bank Management System (Python)

A simple Python project that simulates basic banking operations using file handling.  
It allows users to create accounts, deposit and withdraw money, and track balances and transactions.

---

## ✨ Features
- Create account with a 6-digit account number
- Deposit money
- Withdraw money
- Check balance
- View transaction history
- Stores all data in text files for persistence

---

## 📂 Project Structure
- `balance_history_of_<account>.txt` → Keeps track of balance updates
- `transaction_history_of_<account>.txt` → Records deposits and withdrawals
- `BMS_using_fileConcept.py` → Main program file

---

## 🚀 How to Run
1. Clone or download this repository.
2. Make sure you have Python 3 installed.
3. Run the script:
   ```bash
   python BMS_using_fileConcept.py
